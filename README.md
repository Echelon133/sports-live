Full-stack application for live text coverage of football matches. 

## Used Technologies

**Back End**: *Java, Spring Boot, Jenkins, PostgreSQL, Kafka, Kubernetes, Docker, Socket.io*

**Front End**: *Next.js, Typescript, TailwindCSS*
